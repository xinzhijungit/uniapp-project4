# 页面嵌入外部网址实现说明

## 概述

统计分析和智能问答页面均通过 HTML `<iframe>` 标签实现嵌入外部网址，实现成本低、开发快速。

---

## 1. 统计分析页面

### 文件路径
[statistics/index.vue](file:///Users/xiaoping/Desktop/智慧西安开发/uniapp-project/src/pages/statistics/index.vue)

### 核心代码

```html
<iframe 
  class="statistics-iframe"
  src="https://aichat.jointpilot.com/front/?tenantName=%E4%B8%8A%E6%B5%B7%E5%B8%82%E5%A7%94%E5%85%9A%E6%A0%A1#/dashboard/manage?uniqKey=1807"
  frameborder="0"
  allowfullscreen
></iframe>
```

### URL 参数说明

| 参数 | 值 | 说明 |
|------|-----|------|
| tenantName | `%E4%B8%8A%E6%B5%B7%E5%B8%82%E5%A7%94%E5%85%9A%E6%A0%A1` | 租户名称（URL编码，解码后为"上海市政协校"） |
| uniqKey | `1807` | 唯一标识 |

## 3. 布局结构

```
┌─────────────────────────────────────────────────────┐
│  TopNav (顶部导航栏)                                 │
├─────────────────────────────────────────────────────┤
│  SideMenu  │  main-content                          │
│  (左侧菜单) │  ┌─────────────────────────────────┐  │
│            │  │     <iframe> 嵌入外部页面        │  │
│            │  │     (统计分析/智能问答)          │  │
│            │  └─────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

### 页面组件组成

| 组件 | 作用 |
|------|------|
| TopNav | 系统顶部导航栏 |
| SideMenu | 左侧菜单导航 |
| iframe | 嵌入外部页面内容 |

## 4. 关键样式技巧

### 隐藏外部页面头部

```css
.ai-chat-iframe {
  width: 100%;
  height: calc(100% + 60px);  /* 高度超出容器60px */
  border: none;
  margin-top: -60px;           /* 向上偏移60px */
  position: absolute;
  top: 0;
  left: 0;
}
```

**设计意图**：通过向上偏移 `60px`，隐藏嵌入页面的顶部导航栏（通常高度约60px），避免与当前系统的 `TopNav` 重复显示，实现视觉上的无缝融合。

## 6. 技术特点总结

| 特性 | 说明 |
|------|------|
| **嵌入方式** | HTML5 `<iframe>` 标签 |
| **目标域名** | `aichat.jointpilot.com` |
| **租户标识** | 通过 URL 参数 `tenantName` 传递 |
| **样式处理** | 通过负 margin 隐藏外部页面头部，实现视觉融合 |
| **交互隔离** | iframe 内外部页面完全独立，无数据交互 |

---

## 7. 优缺点分析

### 优点
- 实现成本低，开发快速
- 无需关心外部页面的技术实现细节
- 外部页面更新无需修改当前系统代码

### 缺点
- iframe 内外页面间的数据通信受限
- 外部页面样式可能与当前系统不完全一致
- 浏览器兼容性问题（部分浏览器对 iframe 有限制）
- SEO 不友好（搜索引擎无法抓取 iframe 内容）
